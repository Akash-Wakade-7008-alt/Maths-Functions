# 📐 Maths Functions Library

A growing collection of mathematical functions implemented in different programming languages as part of my learning journey.

This repository is built with a simple idea:  
**understand mathematical concepts deeply by translating them into code.**

---

## 🚀 About the Project

While learning mathematics, many formulas feel abstract at first.  
This repository is my way of turning those formulas into something concrete and logical through programming.

Instead of just solving problems, I focus on:
- Converting formulas into reusable functions  
- Understanding the reasoning behind each concept  
- Strengthening problem-solving skills through implementation  

---

## 🧠 Concepts Covered

This repository includes implementations of concepts such as:

- Mean, Median, Mode  
- Probability Mass Functions (PMF)  
- Distributions (e.g., Poisson Distribution)  
- Algebraic formulas  
- Statistical computations  

More concepts will be added continuously as I progress.

---

## 🛠️ Tech Stack

- Java (current primary language)  
- JavaScript (learning & expanding)  
- Additional languages may be added over time  

---

## ✍️ Documentation Philosophy

Each function is written with clarity in mind and includes:

- Explanation of the mathematical concept  
- Formula reference (inside comments)  
- Input and output details  
- Clean and readable implementation  

The aim is to make every piece of code **educational, not just functional**.
